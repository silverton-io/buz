The Google Pub/Sub sink writes `valid` and `invalid` events to the configured topics.


## Sample Google Pub/Sub Sink Configuration

```
sinks:
  - name: googd
    type: pubsub
    deliveryRequired: true
    project: silverton
    validTopic: honeypot-valid
    invalidTopic: honeypot-invalid
```
